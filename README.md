# Crime_Dataset_India
Analysis of Crime_Dataset
Project Overview:
Analyzed and visualized a real-world crime dataset of 40,000+ records using PostgreSQL and Power BI to uncover insights and trends in criminal incidents.

Tools & Technologies:

Database: PostgreSQL

Visualization: Power BI

Skills: SQL, Data Cleaning, Data Transformation, Analytical Thinking, Dashboard Design

Key Tasks & Achievements:

Loaded and cleaned raw crime data efficiently in PostgreSQL

Created a cleaned view to extract meaningful metrics (e.g., hourly trends, case closure status, police workload)

Built an interactive Power BI dashboard with:

Hourly crime trend visualization

Cases handled per police unit

Case closure analysis

Most frequent crime types

Optimized queries using indexes and aggregation for faster performance

Insights Gained:

Peak hours for crimes

Distribution of cases per police unit

Closure efficiency and pending case metrics

Most common crime types
